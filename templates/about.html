System Overview

This system is designed to help banks predict which customers are likely to leave (churn). It uses machine learning algorithms to analyze customer data and generate predictions to assist in making strategic retention decisions.

Objectives

  • Predict customer churn accurately using machine learning.
  • Provide actionable insights for customer retention.
  • Improve customer relationship management.

How It Works

The system collects relevant customer information such as age, account balance, tenure, number of products, and credit score. Using a trained machine learning model, it predicts the likelihood of the customer leaving the bank.

Key Features

  • Easy-to-use customer data input form.
  • Real-time churn prediction.
  • Dashboard for managing and viewing predictions.
  • Secure login and logout functionality.

Technologies Used

HTML, CSS (Tailwind), and JavaScript for the frontend. Python and machine learning libraries like scikit-learn or TensorFlow for churn prediction.

Importance

Customer retention is critical in the banking sector. This system helps banks proactively retain customers, saving costs and improving satisfaction.

Future Improvements
